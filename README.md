## 🔐 Overview

**Antivirus_v2** is a modular, Python-powered cybersecurity engine designed to detect, analyze, and respond to threats using a combination of:

- 🔍 Signature-based detection  
- 🤖 Machine learning analysis  
- 👁️ Real-time monitoring  
- 🌐 VirusTotal integration  
- 🧩 Extensible architecture  

A lightweight brain with heavy-duty reflexes.

---

## ⚡ Features

- 🧠 Intelligent threat detection  
- ⚡ Fast file & directory scanning  
- 🔬 ML-based classification  
- 👁️ Real-time file system watcher  
- 📊 Logging & analysis engine  
- 🖥️ GUI interface  
- ☁️ Cloud scanning via VirusTotal  
- 🔧 Modular & scalable design
